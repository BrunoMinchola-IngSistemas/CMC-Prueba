CMC Contratistas - Web informativa

Archivos incluidos:
- index.html: página principal con inicio, nosotros, servicios, proyectos, cotización y contacto.
- libro-reclamaciones.html: página independiente para libro de reclamaciones.
- css/styles.css: estilos responsive.
- js/main.js: menú móvil y formularios demo.

Datos por reemplazar:
- Teléfono real
- Correo real
- Dirección real
- Fotos reales de obras
- Logo real, si existe

Nota técnica:
Los formularios no envían datos aún. Están preparados visualmente. Para producción conviene conectarlos a Formspree, Netlify Forms, WhatsApp o un backend propio.
